# ScriptsToolkit 🧰
A curated collection of reusable scripts, environment files, and utilities for bioinformatics workflows.  
Designed to make everyday data analysis and pipeline development faster, cleaner, and reproducible.

---

## 🧬 Repository Overview

**ScriptsToolkit** is my central repository for scripts and helper tools that are frequently used across multiple bioinformatics projects — from bulk RNA-seq to long-read data analysis.
Note: THe scripts are all desgined for submitting jobs to SLURM job schedulers using sbatch. That can be removed for usage directly from the command line
